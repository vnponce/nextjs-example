Project implementing:
- Routing with page.tsx
- `<Image />` component
- `<Link />` component
- Seed data to vercel database
- Request data from vercel database from server component
- Skeletons with `<Suspense />` to avoid waterfall requests
- Invoice
  - Create
  - List
  - Update
  - Delete
- Error handling
